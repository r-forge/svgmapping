Tutorials

 

This section contains step-by-step tutorials to learn the very basic usage of SVGMapping. It is assumed that you know how to use a vectorial drawing software like Inkscape to make templates. In practice you are not stick to Inkscape, you can create templates with any software compatible with the SVG recommendation. Still in this tutorials section we highly recommend to use Inkscape.
